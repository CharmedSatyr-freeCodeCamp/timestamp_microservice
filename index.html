CharmedSatyr's Timestamp Microservice

Type a Unix timestamp or natural language date at the end of this website's URL!

This application will use Node.js, Express.js, and other libraries to return a JSON object in response to your entry.

The object will include both the Unix timestamp and expanded natural language format for your entry. (Unparsable entries will return null values.)


For example:

The entries

https://charmedsatyr.herokuapp.com/December%2015,%202015
https://charmedsatyr.herokuapp.com/1450105200

and

https://charmedsatyr.herokuapp.com/15 dec 2015

all return

{"unix":"1450105200","natural":"Tuesday, December 15th, 2015, 12:00:00 am"}

Try it out!
